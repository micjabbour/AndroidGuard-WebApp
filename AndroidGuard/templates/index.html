Welcome to AndroidGuard

You can use AndroidGuard to track your android device's location, and trigger some remote actions in case it is stolen.

Track mobile device location

The device will send its location periodically to the server. You can find where your devices are when you log in.

Steath mode

Activate stealth mode to hide the application from the home screen.

By doing that, You can only launch it by dialing the secret number, or using remote commands.

Trigger remote commands

You can also trigger remote commands from the website(wipe SD card, refresh location, ...).

SMS commands

In case the device does not have an active internet connection, You can send commands to the device using SMS messages.

The device will reply back with a response to the same number that initiated the command.

currently available remote SMS commands:

  • androidguard getloc

    • To get current location.

  • androidguard wipesdcard

    • To wipe sd card.

  • androidguard showapp

    • To show the app when it is operating in stealth mode.
